What?

The JavaScript Code Retreat is a more focused code retreat. We try to focus even more on the craftmans ship, even more on one language in order to dive even deeper and find out more about how to become better.

How?

Pair programming is at it's core. Sitting in a room with many people so you can switch pairs makes it even more fun. 5 x 45 minutes and one quick retrospective after each allows everyone to share and learn from each other. Yes, one full day of programming just for fun, and throw it away afterwards. But keeping the learnings!
